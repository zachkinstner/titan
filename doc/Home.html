Titan is a distributed graph database optimized for storing and querying massive graphs represented over a cluster of machines. The cluster can elastically scale to support a growing dataset and user base. Titan has a pluggable storage architecture which allows it to build on proven database technology such as Apache Cassandra, Apache HBase, or Oracle BerkeleyDB. Furthermore, the pluggable indexing architecture supports ElasticSearch and Lucene. Titan is a native Blueprints enabled graph database and as such, it supports the full TinkerPop stack of technologies.
